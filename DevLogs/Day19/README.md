\# Project Q 개발일지 - Day19



\## 개발 주제



보스 전용 탄막 패턴 구현



\## 개발 목표



19일차의 목표는 18일차에서 구현한 BossBattle 기본 구조를 확장하여, 보스가 실제 전용 공격 패턴을 사용하도록 만드는 것이었다.



18일차에는 BossBattle 노드 진입 시 보스 1마리가 생성되고 Boss HP Bar가 표시되는 구조까지 구현하였다.  

19일차에는 보스가 단순히 존재하는 적이 아니라, 일정 시간마다 탄막 패턴을 사용하는 보스 전투 형태가 되도록 수정하였다.



\## 구현 내용



\### 1. BossController 스크립트 추가



보스 전용 행동을 담당하는 BossController 스크립트를 새로 추가하였다.



BossController는 보스가 일정 시간마다 공격 패턴을 실행하도록 관리한다.  

패턴은 순서대로 반복되며, 보스 체력이 50% 이하가 되면 패턴 간격이 짧아지도록 구성하였다.



\### 2. BossBullet 스크립트 추가



보스 탄환 전용 스크립트인 BossBullet을 추가하였다.



BossBullet은 지정된 방향으로 이동하며, 플레이어와 충돌하면 PlayerStats의 TakeDamage 함수를 호출하여 피해를 준다.  

일정 시간이 지나면 자동으로 삭제되도록 lifeTime 구조도 함께 구현하였다.



\### 3. BossBullet 프리팹 생성



보스가 발사할 전용 탄환 프리팹을 생성하였다.



BossBullet 프리팹에는 다음 컴포넌트를 연결하였다.



\- BossBullet

\- CircleCollider2D

\- Rigidbody2D



CircleCollider2D는 Is Trigger로 설정하였고, Rigidbody2D는 Kinematic으로 설정하여 탄환 충돌 처리가 가능하도록 구성하였다.



\### 4. BossFirePoint 추가



Boss\_Test 프리팹 안에 BossFirePoint 오브젝트를 추가하였다.



BossFirePoint는 보스 탄환이 생성되는 기준 위치로 사용된다.  

보스 중심보다 약간 아래쪽에 배치하여, 탄환이 보스 몸체 아래에서 발사되는 것처럼 보이도록 설정하였다.



\### 5. 원형 탄막 패턴 구현



보스 주변으로 여러 개의 탄환을 360도 방향으로 발사하는 원형 탄막 패턴을 구현하였다.



이 패턴은 보스 주변 전체를 압박하는 용도로 사용되며, 플레이어가 보스와의 거리를 조절하도록 만드는 역할을 한다.



\### 6. 부채꼴 탄막 패턴 구현



플레이어 방향을 기준으로 좌우로 퍼지는 부채꼴 탄막 패턴을 구현하였다.



이 패턴은 플레이어의 현재 위치를 기준으로 발사되므로, 플레이어가 한 위치에 오래 머무르지 않도록 압박하는 역할을 한다.



\### 7. 조준 연사 패턴 구현



플레이어 위치를 향해 여러 번 탄환을 연속 발사하는 조준 연사 패턴을 구현하였다.



이 패턴은 플레이어가 계속 이동하도록 유도하며, 단순 회피가 아니라 이동 방향을 신중하게 선택하게 만드는 역할을 한다.



\### 8. 보스 체력 기반 패턴 변화 준비



EnemyHealth의 currentHealth와 maxHealth를 읽어, 보스 체력이 50% 이하가 되면 패턴 실행 간격이 짧아지도록 구성하였다.



현재는 패턴 종류 자체가 바뀌지는 않지만, 이후 보스 페이즈 확장 작업에서 체력 구간별 패턴을 추가하기 쉬운 구조가 되었다.

\## 테스트 내용



\### 보스 패턴 실행 테스트



BossBattle에 진입한 뒤, 보스가 일정 시간마다 탄막 패턴을 사용하는지 확인하였다.



\### 원형 탄막 테스트



보스 주변 360도 방향으로 탄환이 발사되는지 확인하였다.



\### 부채꼴 탄막 테스트



플레이어 방향을 기준으로 탄환이 퍼져 나가는지 확인하였다.



\### 조준 연사 테스트



플레이어 위치를 향해 보스 탄환이 연속으로 발사되는지 확인하였다.



\### 플레이어 피격 테스트



보스 탄환이 플레이어와 충돌했을 때 Shield 또는 HP가 감소하는지 확인하였다.



\### 보스 체력 50% 이하 테스트



보스 체력이 절반 이하가 되었을 때 패턴 간격이 빨라지는지 확인하였다.



\## 완료 결과



19일차 작업을 통해 보스가 단순한 큰 적이 아니라, 전용 탄막 패턴을 사용하는 보스 전투 형태로 발전하였다.



이제 BossBattle 노드는 보스 생성, Boss HP Bar 표시, 보스 전용 탄막 패턴까지 포함하는 전투로 구성되었다.  

이를 통해 Project Q의 1챕터 마지막 전투에 필요한 기본 보스전 구조가 마련되었다.
